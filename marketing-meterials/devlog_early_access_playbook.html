01

Product Narrative

One-liner

devLog helps builders plan, track, and share projects — with AI agents that can update logs, todos, and roadmap automatically.

What problem are we solving?

🌀 Builders lose context across commits, chats, TODO files, screenshots, and memory.
✍️ Build-in-public takes effort because progress updates need to be written manually.
🤖 AI coding agents help ship code, but project history and decisions aren't captured in a clean timeline.
🏋️ Project management tools are too heavy for solo builders; GitHub shows code, not the story.

What devLog becomes

🧠 A living project memory layer for each project.
🗺 A clean roadmap with milestones and todos.
🎞 A cinematic timeline of progress logs and decisions.
🌐 A public proof-of-work page for makers.
🔗 A bridge between AI agents and project documentation through MCP + REST.
